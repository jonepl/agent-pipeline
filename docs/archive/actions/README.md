These files are the GitHub Actions implementation of the Agent Pipeline (mode B): four workflow YAMLs (`derive-graph`, `draft-plan`, `implement`, `verify`) and their corresponding `mts` entrypoint scripts. They were deferred after Docker-in-CI authentication proved intractable for a solo dev setup; the pipeline has pivoted to a local always-on polling loop (mode A) as the primary execution model. The Actions approach remains architecturally sound and is preserved here for a future phase when cloud-CI execution is needed.
